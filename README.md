# Angular-GraphQL-Express-Demo

## Dependencies
Angular
Express
Mongoose
Apollo

## Install

Install npm modules.

```shell
npm run app-install
```

copy `.env.example` to `.env`, edit database info, etc.

## Configure Database

Create a table called `users` including two fields `_id` and `username`

Add some mock data manually

## Build & Test

### Client

```shell
npm run start
```

Open `localhost:4200`

### Server

```shell
cd client && ng build
cd server && npm run start
```

Open `localhost:3000`
